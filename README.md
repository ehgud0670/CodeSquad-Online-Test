# CodeSquad-Online-Test

## 1단계 : 간단 야구 게임 구현하기

### 구현해야할 기능 

  * 야구 게임을 진행할 팀 생성  
  * 프로그램이 시작되면 게임시작 메시지 출력
  * 랜덤하게 스트라이크 / 볼 / 안타 / 아웃 네 가지 중 한 결과가 출력
  * 아래 줄에 누적된 스트라이크(S), 볼(B), 아웃(O) 상황을 출력
  * 현재 전체 안타 수 출력 **(자유롭게 구현한 부분 입니다.)** 
  * 스트라이크인 경우 
    * 스크라이크 카운트 1 증가
    * 스트라이크 3회 누적되면 1아웃되게 하기
  * 볼인 경우
    * 볼 카운트 1 증가
    * 볼 4회 누적되면 1안타 되게 하기 
  * 안타인 경우
    * 안타 카운트 1 증가
    * "다음 타자가 타석에 입장했습니다." 메시지 출력
    * 경기 진행되고 다음 타자 차례가 됨
  * 아웃인 경우 
    * 아웃 카운트 1 증가
    * 3 아웃이 아니라면 , "다음 타자가 타석에 입장했습니다." 메시지 출력
    * 3 아웃이 아니라면 , 경기 진행되고 다음 타자 차례가 됨
  * 다음 타자인 경우
    * 다음 타자의 차례에서 현재의 안타, 아웃 카운트는 유지되고, 스트라이크와 볼 카운트는 초기화되게 하기
  * 3 아웃인 경우 게임 종료하기 
  * 게임 종료할 때 **최종 전체 안타수** 출력, "GAME OVER" 메시지 출력
